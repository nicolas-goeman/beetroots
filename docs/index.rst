Beetroots (BayEsian invErsion with spaTial Regularization of nOisy multi-line ObservaTion mapS) is a Python package that performs Bayesian inference with the sampling algorithm described in [1].


============
Installation
============

To perform an inversion, we recommend installing the package.
The package can be installed with ``pip``:

``pip install beetroots``

To reproduce the results from [1], clone the repo with

``git clone ``

Alternatively, you can also download a zip file.
